=============
CrateDB guide
=============

CrateDB is a distributed SQL database that makes it simple to store and analyze
massive amounts of machine data in real-time.

.. NOTE::

   This is a high-level user guide.

   Check out `Getting Started With CrateDB`_ for beginner documentation.

   Check out the `CrateDB Reference`_ for low-level documentation, including
   release notes, configuration, syntax reference, and so on.

.. SEEALSO::

   The CrateDB Guide is an open source documentation project and is `hosted on
   GitHub`_.
